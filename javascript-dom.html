Explain the difference between accessing data from arrays and objects.

- Objects shows a special data type that is alterable/mutable and can be used to store a collection of data instead of a single value. Arrays are a special type of variable that can also be altered and be used to store a list of values.

When to use arrays:
Arrays store the data in an ordered collection in which the data can be accessed using a numerical index.

When to use objects:
Objects are used to represent a ‘Thing’. This could be anything like people, houses, cats etc.

To summarise: If you have to store the data in order or in a sequence then use array otherwise you can use object for everything else.

Explain what functions are and why they are useful.

- Functions are a set of statements (a block of code in JS) that performs a task or calculates a value. They allow you to define a block of code, give it a name and then execute it as many times as you want! Pretty cool. You can define it using function keyword, followed by a name, followed by parentheses ().
Here is an example:

function myFunction(p1, p2) {
return p1 + p2;
}
// The function will return the sum of p1 and p2.
